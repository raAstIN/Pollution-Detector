# سیستم تشخیص وضعیت هوا (Pollution Detector)

این یک برنامه دسکتاپ ساده و مدرن است که با زبان پایتون و کتابخانه `customtkinter` توسعه داده شده است. این ابزار به شما کمک می‌کند تا داده‌های مربوط به آلودگی و رطوبت هوا را تحلیل کرده و وضعیت کیفی هوا را به صورت بصری مشاهده کنید.

## ویژگی‌ها

- **رابط کاربری گرافیکی (GUI):** استفاده از تم مدرن و خوانا.
- **تحلیل زنده (Live Analysis):** با تایپ کردن یا تغییر اعداد در باکس متنی، نتایج بلافاصله به‌روزرسانی می‌شوند.
- **محاسبه خودکار:** محاسبه میانگین آلودگی و رطوبت.
- **نمایش وضعیت:** نمایش وضعیت نهایی هوا با رنگ‌های سبز (سالم)، زرد (هشدار) و قرمز (خطرناک).
- **بارگذاری فایل:** امکان انتخاب فایل متنی (`.txt`) حاوی داده‌ها.

## پیش‌نیازها

برای اجرای این برنامه نیاز به نصب پایتون (Python 3.x) دارید.

## نصب

1. **نصب کتابخانه‌ها:**
   تنها کتابخانه جانبی مورد نیاز برای این پروژه `customtkinter` است. با دستور زیر آن را نصب کنید:

   ```bash
   pip install customtkinter
   ```

2. **نصب فونت‌ها:**
   لطفاً فونت‌های موجود در دایرکتوری `fonts` را نصب کنید تا برنامه با ظاهر صحیح نمایش داده شود.

## نحوه اجرا

برای اجرای برنامه، دستور زیر را در ترمینال یا خط فرمان (در پوشه پروژه) وارد کنید:

```bash
python pollution_detector.py
```

## فرمت داده‌ها

برنامه داده‌ها را خط به خط می‌خواند. هر خط باید شامل سه بخش باشد که با کاراکتر `|` جدا شده‌اند:
`شناسه | درصد آلودگی | درصد رطوبت`

**مثال محتوای فایل:**
```text
1 | 10% | 50%
2 | 55% | 20%
3 | 80% | 60%
```

*نکته: خطوطی که با عدد شروع نشوند یا فرمت صحیح نداشته باشند، به صورت خودکار نادیده گرفته می‌شوند.*
